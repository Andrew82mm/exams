1) в | a => НОД( a ,b ) = b
Док-во: 
{Делители a,b} = {Делители b}
с | a , c | b => c | b.
b | a, c | b => c | a.
Наибольший элемент {Делители a,b} = (a; b)
Наибольший элемент {Делители b} = b
2) a = b * q + c => (a, b) = (b, c);
